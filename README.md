# 实验环境搭建

## 虚拟机

ubuntu16.04


## 编译环境

1. 更新apt软件源
```
apt-get update
```

2. 安装c/c++编译环境
```
sudo apt-get install build-essential
```

3. 补充gcc的完整环境
```
sudo apt-get install gcc-multilib
```
